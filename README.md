# MERN_Learning
MERN_Learning is a beginner-level learning repository created by TheWolf313 for studying and practicing MERN stack development. The project contains basic HTML pages, stylesheets, and JavaScript files organized in folders, serving as a foundational practice ground for full-stack web development.

*Main Function Points*
Learning resource for MERN (MongoDB, Express, React, Node.js) stack development
Contains multiple HTML pages (index.html, index2.html, social.html) for practice
Includes styling and JavaScript functionality examples
Organized folder structure for different learning components
